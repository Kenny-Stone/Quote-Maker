# A QUOTE MAKER

### DESCRIPTION
This script makes a request to pexels using an `api_key` and get's videos based on a query
and then makes another request to a quote website and get's the quotes and then using `moviepy` joins the video and the quote together

#### HOW TO SETUP
- Get dependencies of the scripts
  ```bash
  pip freeze > requirements.txt
  ```
- Install dependencies
  ```bash
  pip install -r requirements.txt
  ```
